# kb/files

## Что хранить: служебные или “прочие файлы”, которые не являются ни шаблонами, ни раздаткой, но нужны боту.

* Типовые варианты:
- картинки/иконки для сообщений (если понадобятся)
- архивы (.zip) с комплектом документов
- вспомогательные файлы для KB (например, keywords.json, stopwords.txt)
- любые вложения, которые ты хочешь давать ссылкой, но не хочешь смешивать с handouts/templates

Правило: “склад всего прочего”, но лучше держать порядок подпапками:
kb/files/images/, kb/files/zips/, kb/files/meta/
